# TECHNICAL ASSIGNMENT

Build an ASP.NET MVC application that consumes NASA's APOD (Astronomy Picture of the Day) API, stores the returned data in a database, and displays the images in a gallery view.

## Prerequisites

*   Ensure the following before starting the assignment:
*   Required Tools & Technologies
    *   .NET SDK
    *   Visual Studio VS Code
    *   SQL Server / SQL Express
*   Basic understanding of:
    *   ASP.NET MVC
    *   DTOs
    *   REST API
    *   JSON serialization/deserialization

## Assignment Requirements

1.  Create a New ASP.NET MVC Project
    *   Set up a clean MVC project structure with Controllers, Models, Views, Services, etc.
2.  Consume NASA APOD API
    *   Use the NASA APOD endpoint to fetch APOD data for a given date range.
        *   NASA Open APIs Please do find the APOD API endpoint from here
    *   The response must be a JSON array which you will deserialize.
